# 纳西妲主题个人作品集网站 PRD

## 项目定位
一个以《原神》角色「纳西妲」为主题的静态相册网站，水平滚动布局，展示约1000张纳西妲图片及二创作品。

## 核心功能

### 1. 首页
- 大标题：如「纳西妲 · 智慧与草木的诗篇」
- 副标题：简短的角色介绍或网站说明
- 进入相册的入口按钮

### 2. 九宫格预览页（过渡页面）
- 点击首页入口按钮后进入
- 全屏展示九张精选照片（3×3布局）
- 每张图片有悬停缩放/光效
- 点击任意图片可进入水平滚动相册
- 退出按钮（返回首页）
- 底部提示：点击图片进入完整相册或点击退出返回首页

### 3. 水平滚动相册（核心页面）
- 横向滚动展示所有图片（类似翻阅画卷/典籍）
- 图片底部显示文件名或简短描述（可选）
- 鼠标悬停有缩放效果或草绿色光效
- 支持键盘左右箭头滚动浏览
- 滚动时显示当前进度/位置

### 3. 图片详情（模态框）
- 点击图片弹出大图展示
- 支持键盘左右键切换上一张/下一张
- 显示图片编号和总数量（如 024 / 1027）
- 关闭按钮（ESC键也可关闭）

### 4. 关于/致谢页
- 网站设计理念说明
- 纳西妲角色介绍（引用游戏内语音或故事片段）
- 图片来源致谢（画师、二创来源链接）
- 技术栈说明（React + Tailwind + GitHub Pages）

### 5. 联系方式/社交链接
- 你的 GitHub 链接
- 其他社交平台（可选）
- 邮箱（可选）

## 可选功能（根据时间决定）
- 背景音乐（须弥地区风格BGM，带开关控制）
- 随机跳转按钮

## 设计要求

### 视觉风格
- **配色**：草木绿（#7cb342）、淡雅金（#ffd54f）、纯白（#ffffff）、深色背景（#0a1f0a 或 #0a0a0a）
- **玻璃态效果**：图片卡片或文字面板使用 backdrop-blur 毛玻璃效果
- **装饰元素**：藤蔓、树叶、花苞、摩斯电码/电路纹样（呼应「智慧」主题）
- **字体**：标题用思源宋体/等宽字体，正文用思源黑体

### 交互体验
- 水平滚动流畅，支持鼠标滚轮横向滚动（需自定义）
- 滚动时有平滑的惯性效果
- 图片悬停有「拉近镜头」的微视差效果
- 页面加载时有优雅的入场动画

### 响应式适配
- 桌面端：完整水平滚动体验
- 平板/手机：支持触摸滑动横向滚动，图片大小自适应

## 技术约束
- 静态网站，托管于 GitHub Pages
- 使用 React + TypeScript + Tailwind CSS
- 图片约1000张，需做懒加载和缩略图优化
- 图片文件名格式：`纳西妲_001.jpg`（已准备好）

## 页面结构（路由）
| 路由 | 页面 | 说明 |
|------|------|------|
| `/` | 首页 | 欢迎页 + 进入相册按钮 |
| `/preview` | 九宫格预览 | 全屏展示9张精选照片，可进入相册或返回首页 |
| `/gallery` | 水平相册 | 核心页面，展示所有图片 |
| `/about` | 关于/致谢 | 网站说明 + 致谢名单 |
| `/contact` | 联系方式 | 社交链接 |

## 非功能性需求
- 首屏加载时间 < 3秒（使用缩略图 + 懒加载）
- 支持现代浏览器（Chrome、Safari、Edge）
- SEO 基础优化（标题、描述、关键词）
- 代码注释清晰，便于后续维护
