# 纳西妲主题个人作品集网站开发指令

## 项目概述
使用 React + TypeScript + Tailwind CSS 开发的个人作品集网站，用于展示约1000张纳西妲（原神角色）图片及二创作品。采用**水平滚动布局**，营造翻阅魔法典籍的沉浸感。

## 开发规范
- 使用函数式组件 + Hooks
- 使用 Tailwind CSS 编写样式
- 组件要可复用
- 代码要有注释
- TypeScript 类型定义完整

## 视觉设计要求
- **配色方案**：草木绿、纯净白、淡雅金为主色调，背景使用深绿色（如 #0a1f0a）或纯黑（#0a0a0a）
- **氛围元素**：可添加藤蔓、树叶、花苞等装饰，按钮/卡片悬停时有草绿色光效
- **玻璃态效果**：使用 backdrop-blur 毛玻璃效果作为图片框架或文字背景
- **字体**：标题使用思源宋体/等宽字体（智慧感），正文使用思源黑体
- **平滑滚动动画**：水平滚动区域需要流畅的滚动体验

## 功能要求
- **水平滚动相册**：打破传统垂直布局，采用横向滚动展示图片
- **图片懒加载**：优化1000+张图片的加载性能，仅加载视口附近的图片
- **响应式设计**：支持移动端触摸滑动（横向滚动）
- **图片模态框**：点击图片弹窗展示大图，支持键盘左右键切换
- **图片编号显示**：每张图片上或底部显示编号（如 纳西妲_001）
- **音乐开关**（可选）：可加入须弥地区风格的背景音乐，有明确的控制开关（使用音乐文件里面的音乐）

## 性能优化要求
- 使用缩略图（宽400-600px）进行列表展示，点击再加载原图
- 图片懒加载（Intersection Observer API）
- 使用 `will-change` 优化滚动性能
- 图片使用 WebP 格式优先（备用 JPG）

## 注意事项
- 保持设计简洁，符合纳西妲“智慧”“草木”“轻盈”的角色特质
- 不要过度设计，确保浏览体验流畅
- 确保所有链接可点击（如社交链接、关于页面等）
- 水平滚动区域不要出现横向滚动条（自定义隐藏或美化）
